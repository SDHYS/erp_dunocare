# ERP 두노케어 - 사용자 매뉴얼

---

## 1. 로그인

- 배포된 사이트 URL에 접속하면 로그인 화면이 나타납니다.
- **관리자 계정** 또는 **팀 계정**으로 로그인할 수 있습니다.
- 관리자 기본 계정: 아이디 `sdadmin` / 비밀번호 `sdadmin@pw`

---

## 2. 관리자 기능

### 2-1. 대시보드 (홈)

- 로그인 후 첫 화면입니다.
- 전체 일정 수, 진행중/완료 건수, 오늘 일정을 한눈에 확인합니다.
- 최근 완료된 일정 목록이 표시됩니다.

### 2-2. 일정 관리

왼쪽 메뉴에서 **일정**을 클릭합니다.

**일정 등록**
1. 우측 상단 **새 일정** 버튼 클릭
2. 날짜, 매장명, 요청사항, 담당팀, 비용 등 입력
3. **등록하기** 클릭

**일정 수정**
1. 일정 목록에서 수정할 일정의 연필 아이콘 클릭
2. 내용 수정 후 **수정하기** 클릭

**일정 삭제**
1. 일정 목록에서 휴지통 아이콘 클릭
2. **확인** 버튼으로 삭제 확정

**일정 필터링**
- 상단 상태 버튼(전체/접수/배정중/진행중 등)으로 필터링
- 검색창에 매장명, 요청사항, 담당자 검색 가능
- 캘린더에서 날짜 클릭 시 해당 날짜 일정만 표시
- **필터 초기화** 버튼으로 전체 보기

**새 팀 등록 (일정 폼 내)**
- 담당자 선택 시 **새 팀 등록** 탭에서 바로 팀 추가 가능

**요청사항 관리**
- 일정 등록 폼의 요청사항 옆 **관리** 버튼 클릭
- 새 요청사항 추가 또는 기존 항목 삭제 가능

### 2-3. 팀 관리

왼쪽 메뉴에서 **팀 관리**를 클릭합니다.

**팀 등록**
1. **팀 등록** 버튼 클릭
2. 팀명(필수), 연락처, 지역, 사업자등록번호, 이메일, 메모 입력
3. 로그인 계정이 필요하면 아이디/비밀번호 설정
4. **등록하기** 클릭

**팀 수정**
1. 목록에서 연필 아이콘 클릭
2. 내용 수정 후 **수정하기** 클릭
3. 비밀번호는 변경할 때만 입력 (빈칸이면 기존 비밀번호 유지)

**팀 삭제**
1. 휴지통 아이콘 클릭 → **확인** 클릭

**팀 상세 보기**
- 팀 행을 클릭하면 하단에 배정 이력(전체/진행중/완료)이 표시됩니다.

### 2-4. 정산 관리

왼쪽 메뉴에서 **정산**을 클릭합니다.

- 모든 일정의 정산 상태를 한 화면에서 관리합니다.
- **정산 상태**: 정산대기 / 정산완료 (이진 상태로 단순화)
- **점주님계산서**: 미발행 / 발행완료
- **협력사 정산**: 미발행 / 발행완료
- 정산완료로 변경하면 **송금일**을 같이 입력합니다.
- **선지급** 필드: 미리 지급한 금액이 있을 경우 입력 — 정산금에서 자동 차감.
- 각 항목의 드롭다운을 변경하면 즉시 저장됩니다.
- 상단에서 상태별 필터링(전체/미정산/정산완료), 검색, 기간(일/주/월) 선택, 팀별/전체 보기 전환이 가능합니다.
- 우측의 **엑셀(.xlsx)** 버튼으로 기존 엑셀 포맷(팀별 시트 + 전체 시트, 정산완료 행 노란색 강조)으로 내보낼 수 있습니다.

#### 정산 타입 (팀 등록 시 설정)
- **직정산 (simple)** — 두노케어 수수료만 차감 + 소득세
- **대행사 경유 (max_care)** — 부가세 → 대행사 수수료 → 소득세
- **복합형 (custom)** — 부가세 → 대행사 → 두노 → 소득세 모든 단계 적용 (1월 맥스케어 경기지사 등 다단계 정산 케이스)

---

## 3. 팀 계정 기능

팀 계정은 관리자가 설정한 아이디/비밀번호로 로그인합니다.

**볼 수 있는 것**
- 본인에게 배정된 일정만 조회 가능

**수정할 수 있는 것**
- 진행상태 (접수/배정중/진행중/진행완료/일정연기/취소)
- 정비요청시간
- 만족도
- 현장담당자
- 비고(메모)

**수정할 수 없는 것**
- 비용, 정산금액, 공제율, 정산상태, 계산서 발행 등 금액 관련 필드
- 다른 팀의 일정

---

## 4. 주의사항

- 브라우저 탭을 닫으면 자동 로그아웃됩니다.
- 24시간 후 세션이 만료되므로 다시 로그인해야 합니다.
- 관리자 비밀번호는 주기적으로 변경하시기 바랍니다.
- 팀 계정의 비밀번호는 관리자가 팀 관리 화면에서 변경합니다.
