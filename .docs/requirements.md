## 🎨 UX 핵심 원칙 ⭐

> **"메인 달력 하나로 대부분의 일을 처리할 수 있게"**

1. **무조건 쉬운 UI/UX** — 대부분 사용자가 어르신
2. **직관성 최우선**
3. **메인 달력 중심** — 모든 작업의 허브
4. **정산/팀 관리 등 부가 기능은 덜 부각되게** — 대표관리자만 사용
5. **엑셀 자동화 필수** — 현재 업체별 따로따로 관리 중인 엑셀을 시스템으로 통합 (수식 자동 계산)

### [5] 정산 관리 — 엑셀 자동화 ⭐ 필수 포함

> 현재 클라이언트가 업체별로 따로따로 관리하는 엑셀 시트들을 **하나의 자동화 시스템으로 통합**

#### 5-1. 업체 등록 (업체명 클릭 시 상세)

- 사업자/프리랜서 선택
- 대표자명
- 주소
- 연락처
- 이메일
- 계좌
- 특이사항
- **정산 규칙 설정** ⚠️ 업체마다 계산식 다름 (아래 5-3 참조)

#### 5-2. 정산 테이블 공통 컬럼

| 컬럼 | 비고 |
|------|------|
| 날짜 | 작업일 |
| 지점명/매장명 | |
| 작업내용 | 자유 텍스트 (예: 배관 스케일링, 에어컨청소 1대, 배관스케일링+세면대 배수구 교체) |
| 총작업비 | 원금액 |
| 개인부품 | 부품 비용 (자재) — 수수료 계산에서 제외되고 정산금에 다시 더해짐 |
| 기사수수료 | |
| 두노케어수수료 | |
| 소득세 (3.3%) | 프리랜서 사업소득세 |
| 정산금 | 최종 지급액 |
| 정산 완료/미비 | 상태 플래그 (송금일 포함) |
| 세금계산서 완료/미비 | 상태 플래그 |

#### 5-3. 엑셀 구조 관찰 (상세 수식은 추후 클라이언트 확인)

엑셀 샘플 7종 분석 결과 공통적으로 아래 흐름을 가짐:

```
총작업비 → (부가세/수수료/소득세 차감) → 정산금
+ 개인부품은 수수료 계산 제외 → 정산금에 다시 가산
```

**중요**: 업체마다 **계산 단계와 차감 비율이 다름** → 업체별 정산 규칙(프리셋) 설정 필요

- 맥스케어 대행사 경유형 (부가세·수수료·소득세 모두 차감)
- 직정산형 (단순 수수료 차감)
- 복합형 (다단계 차감, 경기지사처럼 최종정산 별도 계산)

> 실제 계산식은 클라이언트와 협의 후 업체별로 설정 가능하게 구현

#### 5-4. 자동화 대체 대상 엑셀 시트 (총 7종)

| 엑셀 | 성격 |
|------|------|
| 설비야 | 배관 스케일링 중심 |
| 클린업 | 에어컨 청소 중심 |
| 김광주 맥스케어 | 월별 통합 정산 (미정산 추적) |
| 다인 | 타공/테아창/아크릴 발주 등 (선지급 케이스 있음) |
| 더 클린 | 에어컨 청소 대량 |
| 일등서울설비 | 배관/수전/하수 설비 |
| 1월 맥스케어 경기지사 | 집기/머신 설치, 복합 정산 |

#### 5-5. 필수 자동화 기능

- ✅ 총작업비 입력 → 나머지 금액 자동 계산
- ✅ 업체별 계산 규칙 프리셋 선택
- ✅ 정산/계산서 상태 플래그 + 색상 하이라이트
- ✅ 월별/업체별 합계 자동 집계
- ✅ 미정산 내역 추적
- ✅ 선지급 처리
- ✅ 송금일 기록
- ✅ 엑셀 내보내기 (기존 포맷 호환)
